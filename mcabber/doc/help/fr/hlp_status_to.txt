
 /STATUS_TO jid online|avail|free|dnd|notavail|away [StatusMessage]
 /STATUS_TO jid message StatusMessage

Envoie l'état demandé à "jid".
Si le jid fourni est ".", le contact sélectionné est utilisé.
Remarque : cet état sera modifié par les commandes "/status" ultérieures. Si vous utilisez l'option d'absence automatique (auto-away), l'état sera modifié également.
Remarque : Le jid peut contenir une ressource (i.e. user@server/resource).
